## Maker's First Comment

Hey Product Hunt! I'm the developer behind BizBrain OS.

Here's the insight that led to this: AI is only as useful as what it knows about your business.

ChatGPT remembers that you like Python. BizBrain OS knows your five active projects, their tech stacks, your client relationships, open action items across all of them, and that your client Tim prefers Slack over email.

That's not memory. That's a brain.

**How it works:**

Install the Claude Code plugin. It scans your machine in about 60 seconds -- discovers your repos, project structures, configs, tech stacks. Then it builds a structured knowledge graph: entities (clients, partners, vendors), projects, relationships, action items.

This knowledge persists across every Claude Code session and compounds over time.

**The compounding is the product.** Day 1, Claude knows your project names. Month 6, Claude anticipates your needs -- which clients need follow-ups, which projects are at risk, what decisions were made last quarter. Year 2, a competitor starting fresh is hopelessly behind. Because context takes time. You can't shortcut two years of accumulated business knowledge.

**My favorite feature -- Entity Watchdog:** Mention "Sarah at Acme wants the deadline moved to March" in any conversation. BizBrain automatically updates Acme's action items, Sarah's interaction history, and links the change to the relevant project. You never asked it to. The brain just listens.

**Why open source and local-first:**
- All data stored as Markdown + JSON on your machine
- No cloud dependency, no telemetry, no phoning home
- Human-readable files -- if BizBrain disappeared, your data is perfectly organized
- AGPL v3 licensed

**Honest limitation:** Requires a Claude Code subscription from Anthropic. This is a plugin, not a standalone product. If that's a dealbreaker, I understand.

But if you're already using Claude Code and want it to actually know your business -- install once and let it compound.

## 5 Key Features for Gallery

**1. 60-Second Machine Scanner**
Install the plugin and it scans your filesystem in about 60 seconds. Discovers repos, project structures, tech stacks, configs, and README files. Builds a map of everything you're working on without you explaining anything.

**2. Persistent Knowledge Brain**
The scanner feeds a structured knowledge graph: projects with tech stacks and status, entities (clients, partners, vendors) with roles and history, action items across all projects, and relationships between everything. Stored as Markdown + JSON. Human-readable. Git-friendly.

**3. Entity Watchdog**
Automatically monitors every conversation for mentions of known entities and updates the knowledge graph in real-time. Mention a client preference, a deadline change, or a project decision -- the brain captures it without being asked.

**4. Compounding Context**
Every session makes the brain smarter. Day 1: project names. Month 1: client preferences. Month 6: anticipates your needs. The context compounds daily and becomes your most valuable business asset.

**5. Open Source & Local-First**
AGPL v3. All data stays on your machine. No cloud, no telemetry, no vendor lock-in. Your business context belongs to you. Back up with git, Syncthing, or any method you choose.
